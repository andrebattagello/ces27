Building new instances and buckets

  1. set up a secret.py file under this package with:

    AWS_ACCESS_KEY_ID AWS_SECRET_ACCESS_KEY

    defaults.py expects to find this secret.py module in this package.

  2. python config_new_aws_account.py

    Config a new keypair and security group

  3. python build_new_ec2_instance.py

    Launch a new EC2 Instance

  4. python build_new_s3_buckets.py

    Create new S3 Buckets

S3 utils

  1. python upload_to_static_s3.py '<my_static_folder>'

    Uploads everything under <my_static_folder> to S3. Warning: Files uploaded are gonna be all public.
